Detailed Description

Classe de joueur virtuel qui contrôle le comportement du maillet adverse lorsqu'il est virtuel.

Le comportement du joueur virtuel est divisé en plusieurs scénarios représentant différentes situations de jeu (par exemple, selon la position et la vitesse de la rondelle). Chaque scénario retourne un déplacement qui convient à la situation, pour ensuite être appliqué au maillet dans le ModeleEtatJeu.
